Este repositorio documenta el desarrollo de ejercicios de procesamiento del lenguaje natural y modelización de texto en entorno R, combinando técnicas clásicas de text mining, aprendizaje automático y modelos de lenguaje avanzados.

El primer bloque se centra en el análisis textual de un dataset compuesto por tuits publicados por la cuenta del Museo del Prado. Tras una fase inicial de limpieza y normalización del texto — eliminando menciones, hashtags, URLs y ruido sintáctico — se realiza un análisis lingüístico basado en tokenización y preservación de estructura oracional. El trabajo incluye identificación de términos frecuentes, extracción de palabras clave mediante métodos como RAKE, PMI, frases nominales y TextRank, así como el análisis de coocurrencias y correlaciones léxicas. A partir de estos resultados se exploran relaciones semánticas, identificación de artistas y exposiciones, filtrado temático de tuits y análisis específico de hashtags. Se incorporan además técnicas de representación semántica mediante embeddings (GloVe), evaluación de similitudes entre términos y visualización en espacio reducido mediante t-SNE.

El segundo bloque aborda la clasificación automática de opiniones sobre alojamientos en Andalucía utilizando un dataset de reseñas textuales. Se construyen modelos de clasificación supervisada basados tanto en la calificación original como en una variable derivada de sentimiento (negativo, neutro, positivo), evaluando escenarios multiclase y binarios. El proceso incluye partición entrenamiento-prueba y comparación de desempeño predictivo.

Finalmente, se explora la aplicación de modelos LLM para mejorar la clasificación de sentimiento. Sobre los casos mal clasificados por el modelo tradicional, se emplea un modelo preentrenado para reevaluar predicciones, analizando discrepancias y limitaciones, con el objetivo de contrastar enfoques clásicos frente a arquitecturas basadas en lenguaje profundo.

El repositorio refleja una aproximación integral al análisis textual, abarcando extracción de conocimiento, modelización predictiva y evaluación comparativa entre paradigmas de NLP.
